# fspeed
The plugin add command to change player's fly/walk speed for minecraft.  
Tested in: 1.8, 1.12

## Usage 
/fspeed {amount}  
/fs {amount}

to change your flyspeed. 

/wspeed {amount}  
/ws {amount}

to change your walkspeed.

amount is must be over than 0.0, and under than 1.0.
